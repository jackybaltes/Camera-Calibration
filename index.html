Pinhole Camera Model

Image plane that is mounted (CCD, CMOS, back of your eye)

In front of the plane (distance \(f\)) there is a lens, modeled as a single point

Light reflected from the objects in front of the lens will enter the camera/eye through the point lens and then strike the sensor array

3D Camera Calibration

Camera transformation is modeled as \(P*T*R\)

Set of points \( [u_i, v_i, x_i, y_i, z_i ] \)

\[ A = \left[ \begin{array}{cccc} a_{11} & a_{12} & a_{13} & a_{14} \\ a_{21} & a_{22} & a_{23} & a_{24} \\ a_{31} & a_{32} & a_{33} & a_{34} \\ a_{41} & a_{42} & a_{43} & a_{44} \\ \end{array} \right] \]

For each point \(p_i\) \[ \begin{array}{c} u_i = \frac{ a_{11}*x_i + a_{12}*y_i + a_{13} * z_i + a_{14}}{a_{41}*x_i + a_{42}*y_i + a_{43}*z_{i} + a_{44}}\\ v_i = \frac{ a_{21}*x_i + a_{22}*y_i + a_{23} * z_i + a_{24}}{a_{41}*x_i + a_{42}*y_i + a_{43}*z_{i} + a_{44}} \end{array} \]

\[ \begin{array}{c} u_i*(a_{41}x_i + a_{42}y_i + a_{43}z_{i} + a_{44}) = a_{11}x_i + a_{12}y_i + a_{13} z_i + a_{14}\\ v_i*(a_{41}x_i + a_{42}y_i + a_{43}z_{i} + a_{44}) = a_{21}x_i + a_{22}y_i + a_{23} z_i + a_{24} \end{array} \] \[ \begin{array}{c} a_{11}x_i + a_{12}y_i + a_{13}x_i + a_{14} - a{41}x_iu_i - a_{42}y_iu_i - a_{43}z_iu_i - a_{44}u_i = 0\\ a_{21}x_i + a_{22}y_i + a_{23}x_i + a_{24} - a{41}x_iv_i - a_{42}y_iv_i - a_{43}z_iv_i - a_{44}v_i = 0 \end{array} \]

Linear Distance Transform

Homogeneous system of linear equations (right side of the equation = 0)

Simple solution a_11 = 0, a_12 = 0, ... a_44 = 0

Find the non-zero solution (nullspace) of this system of linear equations

Solution is not unique. Select solution where the norm = 1

Linear Distance Transform

12 unknowns \(a_{11}, ..., a_{14}, a_{21}, ..., a_{24}, a_{41}, ..., a_{44} \)

Need a minimum of 6 points, since each point gives us 2 equations

System may be overconstrained when we have more points

Find least squared error approximation

Penrose-Moore pseudo inverse method (solve \(A^T A = A^T b \) )

Singular value decomposition (SVD)

Singular Value Decomposition

A matrix \(M\) is decomposed into the product of three matrices \( M = U \Sigma V^T \)

\(U\) is a unitary matrix of Eigenvectors of \(MM^T\)

\(\Sigma\) is a diagonal matrix of the square root of the Eigenvalues of \(M^TM\) and \(MM^T\)

\(V\) is a unitary matrix of Eigenvectors of \(M^TM\)

Solution with the smallest Eignvalues for \(M^TM\), last column in \(V^T\) or last row of \(V\)

def calibrateCamera( img, points, normalize = False ):
    A = []
    if normalize:
      iheight, iwidth, _ = img.shape
    else:
      iheight, iwidth = 0, 0

    for p in points:
      uc, vc, x, y, z = p
      u = uc - iheight/2
      v = vc - iwidth/2
      r1 = [ x, y, z, 1, 0, 0, 0, 0, -x * u, -y * u, -z * u, -u ]
      A.append(r1)
      r2 = [ 0, 0, 0, 0, x, y, z, 1, -x * v, -y * v, -z * v, -v ]
      A.append(r2)

    A = np.array( A )
    U,S,Vt = np.linalg.svd(A)
    sol=Vt[-1,:].reshape( (3,4) )
    return sol

sol = calibrateCamera( img, points )

def worldCoordsToImageCoords( A, x, y, z ):
    pt = np.array( [ x, y, z, 1 ] )
    ptd = A.dot( pt )
    
    u = ptd[0]/ptd[2]
    v = ptd[1]/ptd[2]
    return u,v
